Website

a website is a group of web pages or a single web page that are connected to the internet, basically there are two types of website static and dynamic, static websites are websites which only have some pages with information and these sites are not connected with database, dynamic websites are websites which are connected to database and performs action that user requests.

WWW

www stands for world wide web, it is a collection of websites and it helps user to access websites or resources on any device through the internet.

HTTP/HTTPs

http stands for hypertext transfer protocol, it is a protocol of sending information or transfering files across the internet. https stands for hypertext transfer protocol secure and it works same as http but in a secure way.

Client Server

a client is a user who requests service and server is like a provider who provides the requested service to the user.

Domain

a domain is a address, it is like a identity of a website, with the help of domain one can easily access website.

Host

a host is a computer that is connected to internet.

Hosting

a hosting is a service that makes your website public on the internet.

FTP

ftp stands for file transfer protocol, it is a protocol that transfer file from one system to another system.

Path

path is a location of file in a system.

Extension

extension is basically type of file name which is used or inserted at the end of file name.

Editor / Vscode

editor is a program which has capability to edit files, for example vs code: it can convert file into any programming language with ease as it is a coding editor.

URL-URI/URN

url stands for unified resource locator and it gives the network location of resource. uri stands for unified resource identifier and it identifies any resource such as image,document etc. urn stands for unified resource name and it gives only name of resource and not the network location.

Address bar

address bar is a bar in a browser where you can search any website by typing name, it will also show you the address or url of the website.

Title bar

a title bar is a bar where the title of particular file or application is displayed on the top of window.
