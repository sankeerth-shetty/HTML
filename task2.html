Difference between HTTP and HTTPS


Http

Http stands for hypertext transfer protocol
It is a protocol that transfer and receive files or information across internet
Http is used to send or receive data if the data is non sensitive, because http is not safe as it is not encrypted
Http uses port number 80


Https

Https stands for hypertext transfer protocol secure

It is also a protocol that transfer and receive files or information across internet but in a safer way

Https is used to send or receive data if the data is sensitive, because https is safe as it is encrypted

Https uses port number 443
